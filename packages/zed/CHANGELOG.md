# Changelog

All notable changes to the Web Components Zed extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.31] - 2024-11-21

### Added
- Initial release of Zed extension
- Language server integration for web components
- Auto-completion for component tags, attributes, events, slots, and CSS parts
- Hover documentation
- Go to definition support
- Diagnostics and validation
- Support for multiple file types: HTML, JSX, TSX, Vue, Svelte, Astro, MDX, PHP, Twig, CSS, SCSS
- Custom Elements Manifest integration
- Enhanced completion labels for web components
- Workspace configuration support

### Features Parity with VS Code Extension
- Full language server protocol support
- Same core functionality as VS Code extension
- Shared language server package for consistency
